# Simon Say's with Alpine.js

This is a little game demo done with [Alpine.js](https://alpinejs.dev/).

## Run Locally

Clone the project

```bash
  git clone https://github.com/npostulart/alpine-simon.git
```

Go to the project directory

```bash
  cd alpine-simon
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
